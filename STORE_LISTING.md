## Detailed Description

**Smart Tab Booker** is a powerful, privacy-focused Chrome extension that helps you save and organize your browsing sessions. Never lose your open tabs again!

### 🚀 Features

**Manual & Auto Backup**
- Save all open tabs to bookmarks with one click
- Schedule automatic backups: Daily, Weekly, Monthly, or Custom intervals
- Set multiple backup times per day (up to 5 times)

**Tab Organization**
- Preserve Chrome tab groups as bookmark folders
- Smart tab selection - choose exactly which tabs to save
- Duplicate URL detection with option to include or exclude

**Convenience**
- Keyboard shortcut for quick backup (Ctrl+Shift+B)
- Customizable shortcut via chrome://extensions/shortcuts
- Auto cleanup - delete old backups automatically

**User Experience**
- Dark mode support
- Available in 18 languages: English, Turkish, German, French, Spanish, Italian, Portuguese (BR), Russian, Japanese, Chinese (Simplified), Korean, Polish, Dutch, Indonesian, Vietnamese, Arabic (RTL), Hindi, Thai

**Privacy First**
- All data stored locally in your browser
- No external servers
- No tracking
- No account required

### 📋 How to Use

1. Click the extension icon to open the popup
2. Select a bookmark folder for your backups
3. Choose which tabs to backup (all selected by default)
4. Click "Backup Now" or use Ctrl+Shift+B for quick backup

### ⚙️ Settings

- **Auto Backup**: Enable automatic scheduled backups
- **Preserve Tab Groups**: Keep your tab groups organized as folders
- **Include Duplicates**: Choose whether to backup duplicate URLs
- **Auto Cleanup**: Automatically delete backups older than X days
- **Multiple Times**: Set multiple daily backup times

### 🔒 Privacy

Smart Tab Booker respects your privacy. All data is stored locally in your Chrome browser. We don't collect, transmit, or store any of your data on external servers.
